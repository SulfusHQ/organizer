## About

Downloads folders turn into graveyards: files with meaningless names (`document(3).pdf`, `IMG_20240512.png`), no organization, and no easy way to know what's safe to delete.



### What this does

1. **Watches your Downloads folder** 
2. **Reads each new file** and figures out what it actually is
3. **Renames and files it** into the right folder automatically
4. **TODO: Periodically checks** which files in your chosen folders haven't been opened in a while, and suggests what to clean up
